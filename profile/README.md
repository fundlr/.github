# Fundlr

**Fund boldly. Build transparently.**
---
Fundlr is a decentralized crowdfunding platform built on Stellar/Soroban.
Milestone-based escrow, on-chain governance voting, and automatic refunds
on failed campaigns — each campaign functions as a mini-DAO. Built for
African creators locked out of platforms like Kickstarter and Patreon.

## Repositories

- [`fundlr-contracts`](https://github.com/fundlr/fundlr-contracts): Soroban smart contracts (Rust)
- [`fundlr-backend`](https://github.com/fundlr/fundlr-backend): API and Horizon integration (NestJS/TypeScript)
- [`fundlr-frontend`](https://github.com/fundlr/fundlr-frontend): Web app (Next.js/TypeScript)
